## ğŸ“‹ How It Works

This project uses **Redux** to manage global UI and app state, while **React Query** handles all server data fetching and caching, reducing boilerplate and improving performance. Fetching and mutation logic becomes declarative and efficient, and data updates automatically synchronize with UI components.

Styled Components allows writing scoped CSS inside the components themselves, leading to maintainable and reusable styles.
